Address lookup component test

This is a standalone test page for the address lookup + violations display component. If it works here, it gets integrated into the scenario pages.

How to test

  1. Type an NYC address in the input below. Try a known-good one like 142 Ocean Pkwy, Brooklyn or your own NYC address.
  2. Suggestions should appear as you type (after ~3 characters). Click one.
  3. The component should query HPD violations for that building and display them below.
  4. Watch the debug panel at the bottom — it shows every API call and response so you can see what's happening.
  5. If something fails, the debug panel will show the error and the component will fall back to a link-out to HPDOnline.

This page is looking up all violations for testing. In the real scenario pages, the component will filter to scenario-specific violations (mold for mold, pests for pests, etc.).

Before you do anything else

Look up your building's violation history.

HPD maintains a public database of every housing code violation ever issued in NYC. Enter your address to see what's on record for your building.

Having trouble? You can also look up your building directly on HPDOnline.
